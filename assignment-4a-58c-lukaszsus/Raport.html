    x_train = x[:-test_size]
    t_test = t[-test_size:]
    x_test = x[-test_size:]
    return t_train, x_train, t_test, x_test

AutoRegressive Model¶

Model autoregresyjny. Zaimplementowany zosta w nieco trudniejszym wariancie, to jest takim, w ktrym predykcja prbki w kroku t + 1 odbywa si na podstawie wartoci predykowanej z kroku t, a nie wartoci prawdziwej. Po predykcji wartoci w kroku t, nie jest ona zastpowana przez warto prawdziw na potrzeby predykcji wartoci t + 1.

In [123]:
class AutoRegressiveModel:
    def __init__(self, window):
        self.window = window
        self.model = None
        self.coef = None
        self.history = None
        
    def fit(self, x_train):
        self.model = AutoReg(x_train, lags=self.window)
        model_fit = self.model.fit()
        self.coef = model_fit.params
        
        # create history of time series for prediction
        history = x_train[len(x_train) - self.window:]
        self.history = [history[i] for i in range(len(history))]
        
    def predict(self, n):
        history = self.history.copy()
        predictions = list()
        for t in range(n):
            length = len(history)
            lag = [history[i] for i in range(length - self.window, length)]
            yhat = self.coef[0]
            for d in range(self.window):
                yhat += self.coef[d + 1] * lag[self.window - d - 1]
            predictions.append(yhat)
            history.append(yhat)
        return predictions
    
    def predict_start_with(self, n, start_with):
        if len(start_with) < self.window:
            raise ValueError("Length of starts_with has to be greater or equal to self.window.")
        history = list(start_with.copy())
        predictions = list()
        for t in range(n):
            length = len(history)
            lag = [history[i] for i in range(length - self.window, length)]
            yhat = self.coef[0]
            for d in range(self.window):
                yhat += self.coef[d + 1] * lag[self.window - d - 1]
            predictions.append(yhat)
            history.append(yhat)
        return predictions

        model = AutoRegressiveModel(window = window_size)

        mae = sqrt(mean_absolute_error(x_test, predictions))
        rmse = sqrt(mean_squared_error(x_test, predictions))

Podsuomowanie¶

Wykorzystujc zbir danych Airline Passengers wykonano predykcj ostatnich dwunastu wartoci szeregu czasowego.

Prace rozpoczto wykonaniem wykresu, pozwalajcym zapozna si z charakterme danych. Stworzono model autoregresyjny (AR) oraz wykonano strojenie hiperparametru tego modelu, ktrym jest liczba opnie (rozmiar) wykorzystana w modelowaniu. Na podstawie przeprowadzonych bada wybrano najlepsz warto hiperparametru. Nastpnie zbadana autokorelacj (ACF i PACF) szeregu czasowego oraz szeregu wygenerowanego za pomoc wyuczonego modelu.

Kierujc si miar Root Mean Square Error, wybrano model o liczbie 22 opnie, jako najlepiej modelujcy szereg czasowy. Wybrano model AR, poniewa wstpna analiza szeregu czasowego pozwolia przypuszcza, e wyrana sezonowo i trend zostan dobrze zamodelowane przez ten model.

Wnioski¶

  • Model autoregresyjny dobrze poradzi sobie z modelowanie zadanego szeregu czasowego.

  • Sezonowo szeregu czasowego wynoszca 12 prbek (miesicy), powoduje e dla k od 1 do 12 nastpuje gwatowny spadek wartoci bdu, a nastpnie stablizuje si on. K oznacza liczb opnie w modelu AR(k) oraz przesunicie w operatorze LAG-k.

  • Badajc autokorelogram szeregu czasowego oraz szeregu wygenerowanego za pomoc modelu AR, mona zauway, e bardzo dobrze modeluje on sezonowo szeregu. Nieco gorzej wypada modelowanie rezyduw. Z tym zadaniem pewnie lepiej poradziby sobie model MA.
